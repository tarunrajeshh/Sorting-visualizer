# Sorting Visualizer

A simple web-based sorting visualizer that demonstrates how different sorting algorithms work using animated bars.

## Algorithms Implemented
- Bubble Sort
- Insertion Sort
- Quick Sort
- Radix Sort

## Features
- Visual representation of sorting
- Step-by-step animation
- Multiple algorithms to compare

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open `index.html` in a browser
